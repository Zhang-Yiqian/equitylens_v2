# AI 美股产业拐点挖掘与决策 SaaS PRD (v1.0 自动宏观增强版)

## 0. 文档信息
* **产品代号**：MVPify-AlphaTracker
* **文档版本**：v1.0 (极简双引擎 + 10-K 宏观增强)
* **核心目标**：构建极简可视化的 SaaS 平台，利用大模型将“基础财务表”、“公司新闻流”与“SEC 10-K 官方宏观画像”进行自动化交叉验证，精准挖掘 6-12 个月维度的业绩催化剂。

## 1. 核心边界与生死假设 (CRITICAL SCOPE)
* **数据极简主义**：剥离高理解成本的筹码数据与卖方评级。底层动态数据锁死在 **Yahoo Finance (基础财务表 + 公司新闻)**，底层静态宏观数据锁死在 **SEC EDGAR (10-K 商业与风险披露)**。
* **生死假设**：大语言模型（LLM）能否单纯依靠冷冰冰的财务异动，结合最新新闻事件的脉络，并在 10-K 划定的竞争格局内，准确推理出“利润表即将迎来阶跃”的不可逆拐点？
* **功能闭环**：Web 界面选择标的 -> API 自动抓取 (财报+新闻+10K片段) -> 大模型多维推理 -> 终端输出标准研判报告。

## 2. V1.0 核心实施模块 (SaaS Engine)

### 模块一：三位一体数据馈送器 (Tri-Core Feeder)
* **优先级**：**P0**
* **功能描述**：后端接收前端传入的 Ticker，自动化执行三大类数据的极简抓取：
    1. **财务 Hard Truth (经由 Yahoo Finance)**：最近 4-8 个季度的核心财务指标（营收、毛利、营业利润、经营现金流、CAPEX）。
    2. **事件 News Feed (经由 Yahoo Finance)**：最近 15-30 天的高相关度公司新闻摘要（标题 + 核心内容）。
    3. **官方宏观底座 (经由 SEC EDGAR API 免费拉取)**：提取该 Ticker 最新一份 10-K 报告中的 `Item 1 (Business)` 和 `Item 1A (Risk Factors)` 纯文本摘要。
* **完成标准 (DoD)**：清洗新闻噪音，提取 10-K 核心段落，将“宏观背景 + 动态财务 + 近期事件”按固定结构组装为 Prompt Input Payload。

### 模块二：宏观锚定下的交叉验证引擎 (The Cross-Validation Engine)
* **优先级**：**P0**
* **功能描述**：注入超级 Prompt，AI 首先读取 10-K 建立宏观认知基线，然后对财务和新闻进行 6-12 个月爆发信号的扫描。AI 需遍历以下矩阵：

**[A] 竞争格局与产业链基线 (基于 10-K 文本)**
1. **护城河与竞品对标**：从 10-K 中识别主要竞争对手与主营业务。后续的财务与新闻分析，必须时刻回答：“这究竟是全行业红利，还是从竞品手里抢到的份额？”
2. **风险因子证伪**：10-K 中披露的核心风险（如客户集中度过高、供应链受制于人），在近期的财务和新闻中，是被恶化了，还是被成功化解了？

**[B] 财务表内部的异动侦测 (基于 YF 财务数据)**
3. **经营杠杆释放**：营收保持增长，但营业费用（OpEx）增速骤降，利润即将爆发。
4. **现金流/CAPEX 剪刀差**：经营现金流转正并大幅覆盖资本支出，进入纯利收割期。
5. **利润率阶跃**：毛利率连续两个季度出现超预期环比拉升。

**[C] 新闻流事件驱动验证 (基于 YF 资讯)**
6. **大单与新品兑现**：新闻中是否证实了财务表上的“研发投入”终于转化为明确的商业化订单？
7. **产业链共振**：结合 10-K 的上下游描述，新闻中提及的产业利好，是否能完美印证该公司的财务改善趋势？
8. **降本增效落地**：裁员、剥离不良资产等新闻事件，作为未来财务模型改善的前置信号。

### 模块三：SaaS Web UI 极简交互 (User Interface)
* **优先级**：**P1**
* **功能描述**：包含两大核心零摩擦交互页面。
    * **全景自选股看板 (Watchlist Dashboard)**：展示 NVDA, APP, PLTR, LITE 等 10 大核心标的的最新价格与系统评级（Buy/Watch/Avoid），支持一键批量异步刷新。
    * **单标的深度卡片 (Ticker Report)**：
        * 展示大模型生成的 Markdown 研判报告。报告强制包含三个段落：**综合结论**、**基于 10-K 竞争格局的财务与事件催化剂分析**、**核心风险提示**。
        * 提供 YF 财务数据趋势可视化折线图（营收、利润率等基础视图）。
