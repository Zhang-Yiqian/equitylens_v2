# AI 美股产业拐点挖掘与决策 SaaS PRD (v0.3 MVP版)

## 0. 文档信息
* **产品代号**：equity_lens
* **文档版本**：v0.3 (核心引擎定型版)
* **核心目标**：从非结构化产业文本中寻找支撑未来 6-12 个月股价爆发的“不可逆盈利拐点”与催化剂。
* **目标受众**：中长线/大波段投资者（一期为创始人自用，终局为 SaaS 用户）。

## 1. 产品愿景与生死假设 (The Make-or-Break)

### 1.1 核心痛点与产品愿景
传统量化工具极度依赖滞后的财务报表，难以捕捉“暴涨前夜”的微观信号。本产品旨在打造一个具备资深买方研究员视角的 AI 引擎，通过深度咀嚼长达数万字的财报电话会议（Earnings Call）与 SEC 披露文本，精准定位跨过“不可逆盈利拐点”的优质标的。

### 1.2 关键范围检查 (CRITICAL SCOPE CHECK)
* **绝对禁止（Out of Scope for MVP）**：泛网络新闻爬虫、前端 Web UI、用户登录/计费系统、全市场自动化批量扫描。
* **生死假设**：大语言模型（LLM）能否在克服长文本“注意力衰减”的前提下，基于 SEC 纯净数据和电话会议文本，利用内置的 12 维度雷达矩阵，准确提炼出 **6-12 个月内可兑现的业绩催化剂**，且不产生事实幻觉？
* **MVP 最小功能闭环**：单标的输入 -> API 抓取财报与会议文字稿 -> 大模型矩阵式推理 -> 终端输出强逻辑 Markdown 研判报告。

## 2. 数据架构与来源 (MVP 阶段限定)
MVP 阶段采用“绝对干净”的高信噪比 API，杜绝爬虫工程消耗。

| 数据类型 | 来源推荐 | 作用与要求 |
| :--- | :--- | :--- |
| **关键财务 Hard Truth** | FMP API / SEC EDGAR | 提取营收、毛利、经营现金流、RPO、CAPEX 等硬核指标。若缺失必须标明，严禁 AI 编造。 |
| **电话会议文字稿 (Transcript)** | FMP API | **系统核心信号源**。包含高管陈述与分析师 Q&A 的全量文本（通常 1-2 万字）。 |

## 3. MVP 核心实施模块 (The MVPify Breakdown)

### 模块一：高信噪比数据组装器 (The High-Signal Feeder)
* **优先级**：**P0**
* **功能描述**：从硬编码的 10 只核心 AI 产业链高弹性标的池中输入特定股票代码（Ticker），通过 API 自动拉取最近一季的财务快照与完整的电话会议文字稿。
* **MVP 默认论证标的池 (Top 10 High-Elasticity AI Universe)**：
  - **[AI 软件与应用层]**
    - `APP` (AppLovin: AI 广告算法与变现龙头)
    - `PLTR` (Palantir: 企业级/政府级 AI 平台)
    - `CRWD` (CrowdStrike: AI 驱动的终端网络安全)
  - **[AI 能源与基建层]**
    - `CEG` (Constellation Energy: 数据中心核能供电)
    - `VST` (Vistra Corp: 数据中心综合天然气与核能电力)
    - `ASTS` (AST SpaceMobile: 卫星通信/太空宽带基建)
  - **[光通信与互联层]**
    - `LITE` (Lumentum: 数据中心光模块与核心器件)
    - `CIEN` (Ciena: 数据中心 DCI 光传输网络)
  - **[硬件与高速存储层]**
    - `WDC` (Western Digital: AI 数据中心大容量企业级存储)
    - `MU` (Micron: HBM 高带宽内存绝对核心)
* **完成标准 (DoD)**：将提取的数据自动拼接为标准的 JSON 或 TXT 上下文载荷，作为大模型的底层 Input。

### 模块二：6-12 个月产业拐点挖掘引擎 (The Inflection Engine)
* **优先级**：**P0** (系统的灵魂护城河)
* **功能描述**：通过注入包含“思维链 (CoT)”的超级 Prompt，强制 AI 忽略“5年后的宏大叙事”，只捕获未来 2-4 个季度内可兑现的利润催化剂。AI 必须遍历以下 **12 维度扩充矩阵**（无用则略过，命中则深度分析并引用原文）：

**[A] 财务先行与高确定性指标 (Leading Indicators)**
1. **RPO/递延收入激增**：客户长单提前锁定，远期业绩爆发明牌。
2. **FCF 转化率拐点**：经营现金流大幅转正，CAPEX 见顶，进入利润收割期。
3. **毛利率阶跃**：高毛利新产品占比提升带来的结构性利润率改善。

**[B] 硬件/半导体专属纵深挖掘 (Hardware & Semi)**
4. **供应链瓶颈解除**：卡脖子环节（如良率、特定零部件）被证伪或突破。
5. **大厂 CAPEX 绑定**：订单深度绑定北美云厂商激进的资本开支计划。
6. **单机价值量提升**：随下一代平台（如新架构 GPU/光模块迭代）实现量价齐升。

**[C] 软件/SaaS 专属纵深挖掘 (Software & SaaS)**
7. **客户 ROI 验证**：具备明确的降本增效量化案例，证明非情绪性炒作。
8. **算力经济学拐点**：大模型推理成本下降带来的底层毛利率显著改善。
9. **旧业务蚕食防御**：确认 AI 新业务带来的是净增量，而非洗劫原有客群。

**[D] 文本情绪与博弈暗号 (Sentiment & Game Theory)**
10. **管理层防御闪躲**：对降价、流失、竞争等尖锐提问的语焉不详与套话掩饰。
11. **指引异常自信**：对未来产能锁定、需求远超供给使用绝对化量词确认。
12. **分析师语气反转**：一贯苛刻的分析师转向祝贺，或提问焦点从“存亡”转向“交付能力”。

* **工程防踩坑**：强制 AI 使用 `<evidence>` 标签，任何研判必须逐字引用电话会议原文片段作为支撑。

### 模块三：多视角研判报告输出 (Report Generator)
* **优先级**：**P0**
* **功能描述**：基于模块二的挖掘结果，输出供专业投资者决策的标准化研判。
* **完成标准 (DoD)**：在终端或本地生成 Markdown 报告，包含：
    * **综合研判结论**（Conviction Buy / Watch / Avoid）。
    * **核心催化剂与原文证据**（6-12 个月内即将发生的业务爆发点）。
    * **证伪风险点**（如果买入后大跌，最可能是哪个环节的假设破裂了？）。
    * **下季度定量追踪指标**（如：必须盯紧下季度的存货周转天数或 NDR）。

## 4. 演进路线图 (Roadmap & Future Iterations)
在 MVP 论证“单只股票的深度挖掘与预判有效性”后，逐步向全自动化 SaaS 拓展。

| 阶段 | 核心模块/功能 | 目标说明 |
| :--- | :--- | :--- |
| **V0.5 (自动化批处理)** | 极简多标的扫描 | 支持触发自选股池里的股票跑一遍，直接看 Markdown 报告。 |
| **V1.0 (SaaS 初版)** | Web UI 与交互面板 | 开发前端页面，展示产业链地图、排行榜、单公司研判详情卡片，支持用户自行输入 Ticker 测算。 |
| **V1.5 (全视野数据源)** | 替代数据融合 | 引入行业新闻 API、关键供应链上下游公司的交叉印证分析。 |
| **V2.0 (终极自迭代)** | AI 回测与权重微调 | 引入 Ground Truth 真实涨幅数据回传，让系统自动评估此前预测的准确率，自主微调 12 维度矩阵在不同行业的权重评分。 |
